# whistle抓包

<https://juejin.cn/post/7140040425129115684?searchId=202405251350148FC5CC866BAD2A71205E>

```
 
npm i -g whistle & w2 start
安装后，可以在电脑上设置全局代理，代理的端口为 8899.

w2 proxy // 设置全局代理

w2 proxy off // 关闭全局代理

就可以通过浏览器访问 http://127.0.0.1:8899/ 查看抓包、修改请求等。
```
