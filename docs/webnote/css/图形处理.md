# 图形处理



## 蒙版mask

可以基于**有透明通道的图片**进行颜色填充

```
mask:url() 50% 50% / cover
-webkit-mask:url() 50% 50% / cover
```



## 褶皱实现 mix-blend-mode

```
mix-blend-mode: multiply; 
```

### "multiply"（正片叠底）

红1 乘 红2 /255

绿1 乘 绿2 / 255

蓝1 乘 蓝2 / 255

"multiply"混合模式将元素的颜色与其下方元素的颜色进行相乘，从而产生一种混合效果。它通过减少亮度来混合颜色，使得较暗的颜色更加突出。这种混合模式通常用于创建阴影、模糊效果或颜色叠加效果。
