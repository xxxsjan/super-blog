## 开发者选项

usb调试

usb安装

数据线连手机

解压

点击exe文件

如果不成功

win x打开终端，cd到解压目录

（地址栏输入cmd也可以，没管理员win x）

执行 ./scrcpy.exe

回车后可以看到失败原因

![image-20230819022249252](https://raw.githubusercontent.com/xxxsjan/pic-bed/main/202308190222394.png)

-d 使用usb

-e 使用wifi

首次连接手机使用usb的话，第二个是读取手机wifi信息，后面断了usb，wifi可以无缝衔接

./scrcpy.exe -d

./scrcpy.exe --tcpip

首次使用wifi的话，第二个是usb连接

./scrcpy.exe --tcpip

./scrcpy.exe -e
